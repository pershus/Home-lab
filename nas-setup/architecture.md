### TrueNAS

I have decided to use trueNAS as the NAS program. This comes from some key points 
1) Open source software 
2) increased data integrity systems (at the cost of more RAM usage), easy to install RAID systems 
3) Flexibility and ease of use.
4) Allows for easy segmentation of users

This does however come at the cost of using quite a bit of RAM, but as the system it is running on is quite beefy (as of now) this is not a major hurdle.
